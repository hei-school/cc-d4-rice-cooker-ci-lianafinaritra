## About The Project

## Installation
Make sure you have a RUBY environment

### How to run

* Clone the project
  ```ssh
  git clone
* Checkout to ruby branch
  ```ssh
  git checkout feature/Ruby
* Run the following command
  ```npm
    ruby src/app.rb

### Linter

* Run the linter
  ```ssh
    rubycorp .
